# Escala de Plantao

Aplicacao local em React para visualizar e ajustar a escala de plantao.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Persistencia

- Servidores e overrides sao salvos somente no `localStorage` do navegador.
- Nao ha dependencias de backend.
